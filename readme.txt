ส่วนที่ 1 โครงสร้างไฟล์ ประกอบด้วย
1_plan/ เป็นโฟลเดอร์จัดเก็บแผนงานหรือโครงสร้างงานก่อนที่จะทำโปรเจ็กต์แบบเต็มรูปแบบ
2_doc/ เป็นโฟลเดอร์เก็บรายงานรูปเล่มในรูปแบบไฟล์ .docx หรือ .pdf และสไลด์นำเสนอในรูปแบบไฟล์ .pptx หรือ .pdf
3_vdo_demo/ เป็นโฟลเดอร์จัดเก็บวิดีโอคลิปนำเสนอโปรเจ็กต์แบบเต็มรูปแบบ 
4_sql/ เป็นโฟลเดอร์เก็บฐานข้อมูลที่ชื่อ swu-air.sql 
5_code/SWU-AIR/ เป็นโฟลเดอร์เก็บไฟล์ทั้งหมดที่จะนำไปอัพโหลดบนเซิร์ฟเวอร์ 
การติดตั้งโครงงาน 
1. ติดตั้งโปรแกรม XAMMP 
2.นำไฟล์ที่อยู่ในโฟลเดอร์ 5_code/SWU-AIR/ มาจัดเก็บไว้ในโฟลเดอร์ htdocs ในที่โฟลเดอร์ที่ติดตั้ง XAMMP ไว้
3. นำสคริปต์ swu-air.sql สำหรับสร้างฐานข้อมูลรันใน PHPMyAdmin เพื่อสร้างฐานข้อมูลและตารางที่เกี่ยวข้อง

การแก้ไข
เพิ่ม foreign key ใน books
+ plane_1 เป็น FK ของ plane_id จาก table planes
+ source เป็น FK ของ stn_code จาก table airports
+ flight_id เป็น FK ของ flight_id จาก table flights

โดยไฟล์ใหม่ทั้งหมดจะมีคำว่า _แก้ ต่อท้ายในโฟลเดอร์ Report และ โฟลเดอร์ Slide
